# Button

This is an example of a button component.

<code-bubble>

```html
<my-button>My Button</my-button>
```

```tsx
import { MyButton } from 'lit-starter-kit/react';

export default () => {
  return <MyButton>My Button</MyButton>;
};
```

</code-bubble>

## Variations

Here are some examples of button variations.

<code-bubble>

```html
<my-button>My Button</my-button>
<my-button variation="primary">My Button</my-button>
<my-button variation="hollow">My Button</my-button>
<my-button variation="transparent">My Button</my-button>
```

```tsx
import { MyButton } from 'lit-starter-kit/react';

export default () => {
  return <>
    <MyButton>My Button</MyButton>
    <MyButton variation="primary">My Button</MyButton>
    <MyButton variation="hollow">My Button</MyButton>
    <MyButton variation="transparent">My Button</MyButton>
  </>;
};
```

</code-bubble>
